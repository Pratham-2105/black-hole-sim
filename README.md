## Introduction
This project simulates black hole behaviors and gravitational effects on nearby objects.

## Features
- Real-time simulations
- Detailed gravitational modeling
- High-quality graphics

## Installation
To install the simulator, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Pratham-2105/black-hole-sim.git
   ```
2. Navigate to the project directory:
   ```bash
   cd black-hole-sim
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To run the simulator, use the following command:
```bash
npm start
```
